## 插件

### emmet-vim

写前端必备的emmet插件

```shell
git clone https://github.com/mattn/emmet-vim.git
cd emmet-vim
mkdir .vim
mkdir .vim/plugin
mkdir .vim/autoload
cp plugin/emmet.vim ~/.vim/plugin/
cp autoload/emmet.vim ~/.vim/autoload/
cp -a autoload/emmet ~/.vim/autoload/
```
